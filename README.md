# InterrupCat (インタラプキャット) 🐾

タイピング中、特定のキーや単語に反応して「鳴き声」や「お好みの音」を奏でる、遊び心満載のChrome拡張機能です。
仕事や執筆の合間に、ちょっとした癒やしと楽しさを提供します。

## ✨ 主な機能

- **タイピング・トリガー**: 指定したアルファベット（例: `c`, `a`, `t`）を打つたびに音が鳴ります。
- **IME変換トリガー**: 「かわいい」などの特定の単語に変換を確定した瞬間に音が鳴ります。
- **自由自在なカスタマイズ**:
  - 好きな音声ファイル（MP3/WAV等）をアップロード可能。
  - ブラウザ上で直感的に波形をカット（トリミング）できるエディタ搭載。
  - 音量調整機能（最大300%までブースト可能）。
  - トリガーとなるキーや単語はいつでも自由に追加・削除できます。
- **プレミアム・デザイン**: X（旧Twitter）風の洗練されたライトテーマUI。

## 🚀 インストール方法（開発者モード用）

開発版としてインストールいただけます：

1. このリポジトリをZIPチャットでダウンロードし、任意のフォルダに解凍します。
2. Chromeブラウザを開き、アドレスバーに `chrome://extensions/` と入力して移動します。
3. 右上の **「デベロッパー モード」** をONにします。
4. **「パッケージ化されていない拡張機能を読み込む」** ボタンを押し、解凍したフォルダを指定します。
5. ツールバーに青い猫のアイコンが表示されれば完了です！

## 🛠 使い方

### 1. 基本設定
- ツールバーの猫アイコンをクリックしてポップアップを開きます。
- 「詳細設定」から設定画面へ移動します。

### 2. 音声のカスタマイズ
- 設定画面の「音声アップロード」エリアに、使いたいMP3や動画などのファイルをドラッグ＆ドロップします。
- 青い波形が表示されるので、鳴らしたい範囲をマウスで選択（トリミング）します。
- 「リストに追加して設定」を押すと、その音が即座に反映されます。

### 3. トリガーの編集
- 設定画面下部の「再生トリガー設定」から、反応させたい「キー（英数字1文字）」や「変換単語」を追加してください。
- 登録済みの項目をクリックすると削除できます。

## 📝 動作デモ
プロジェクト内の `demo.html` をブラウザで開くと、大きな文字でタイピング音を確認できる専用のデモ画面を利用できます。
