# 🐾 InterrupCat (インタラップ・キャット)

タイピングや IME 変換確定に合わせて、愛らしい猫の鳴き声（またはお好みのサウンド）を届ける Chrome 拡張機能です。
作業中の静寂にささやかな癒やしと楽しさをプラスします。

## ✨ 特徴

- **タイピング連動**: 特定のキーを打つたびに音が鳴ります。
- **IME 確定トリガー**: 日本語入力で特定の単語を確定した瞬間に鳴らすことができます。
- **ランダム再生**: 複数の音声を選択して、タイピングのたびにランダムな音色を楽しめます。
- **なめらかな音響**: フェードイン・フェードアウト機能により、ぶつ切り感のない自然なサウンドを実現。
- **専用エディタ**: ブラウザ上で音声ファイルのトリミング、音量、フェード秒数を直感的に設定。
- **アイコン＆ネーミング**: 音声ごとに好きな名前とアイコン（画像）を設定可能。X（旧Twitter）風の切り抜きエディタで、自分だけのサウンドリストを作れます。
- **Playground (テストエリア)**: 保存した設定を、その場ですぐに試せるテスト入力欄を完備。

## 🚀 インストール方法（開発者モード用）

現在はストア公開前の開発版としてインストールいただけます：

1. このリポジトリをダウンロードし、任意のフォルダに解凍します。
2. Chrome ブラウザを開き、アドレスバーに `chrome://extensions/` と入力して移動します。
3. 右上の「デベロッパーモード」を ON にします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、解凍したフォルダを選択します。

## 🛠 使い方

### 1. 基本設定
- パズルピースのアイコン（拡張機能メニュー）から **InterrupCat** をピン留めし、アイコンをクリックします。
- 「機能の有効化」スイッチで ON/OFF を切り替えられます。
- 「詳細設定」から、より細かなカスタマイズが可能な設定画面へ移動します。

### 2. 音声のカスタマイズ
- 設定画面の「音声アップロード」エリアに、使いたい MP3 や動画などのファイルをドラッグ＆ドロップします。
- 青い波形が表示されるので、鳴らしたい範囲をマウスで選択（トリミング）します。
- 音量、フェードイン、フェードアウトの秒数を調整し、「選択した音を追加」を押します。
- 表示されるモーダルで**音声の名前**と**アイコン画像**を設定して保存すると、リストに追加されます。
- リストが増えすぎた場合は、「リストをリセット」ボタンで初期状態に戻すことも可能です。

### 3. トリガーの設定
- 指定した特定のキー（a, b, c... 等）を打った時に鳴らすか設定できます。
- 日本語入力で「かわいい」と確定した時に鳴らす、といった単語トリガーも登録可能です。

### 4. Playground でテスト
- 設定画面の一番下にある **Playground** の入力欄に文字を打つと、その場で現在の音やトリガー設定を試せます。

## 📝 動作デモ
プロジェクト内の `demo.html` をブラウザで開くと、大きな文字でタイピング音を確認できる専用のデモ画面を利用できます。
